此專案用於搶711賣貨便，需由前半段的notebook開啟瀏覽器後自行登入，並於notebook內設定商品關鍵字、規格、賣家指定資訊等等，即可於賣場開放時自動填入表單，並送出訂單，加速整體購物流程。
Requirements: Selenium
Optional: openCV, matplotlib
